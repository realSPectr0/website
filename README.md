# realSPectr0 — Personal Website

A personal portfolio website for showcasing my cybersecurity journey, projects, CTF/lab writeups, technical notes, and contact information.

This site is built as a modern developer portfolio with a clean dark theme and sections for projects, skills, writeups, and personal information.

## Live Site

[View Website](https://temp3st.vercel.app/)

## About

This website is my personal cybersecurity portfolio. It highlights my work in:

- Cybersecurity
- CTFs and lab practice
- Web security
- Linux and networking
- Programming projects
- Technical writeups
- OSCP-style learning

## Features

- Modern responsive portfolio design
- Dark, minimal developer-style UI
- Project showcase section
- Cybersecurity lab/writeup content
- Configurable personal information
- Reusable React/Next.js components
- TypeScript-based codebase
- Vercel-ready deployment

## Tech Stack

- Next.js
- React
- TypeScript
- CSS
- Vercel

## Project Structure

```text
.
├── app/                  # Main Next.js app pages and layouts
├── components/           # Reusable UI components
├── config/               # Site configuration
├── content/lab-writeups/ # Cybersecurity lab/writeups content
├── data/                 # Portfolio data such as projects and links
├── hooks/                # Custom React hooks
├── lib/                  # Utility functions
├── public/               # Static assets and images
├── styles/               # Global styles
├── types/                # TypeScript types
└── schemas/              # Content/data schemas
```
